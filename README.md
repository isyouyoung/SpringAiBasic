# 🤖 SpringAiBasic

> Spring Boot를 활용하여 AI 및 OCR(광학 문자 인식) 기능을 학습하기 위해 만든 실습 프로젝트입니다.

---

## 📌 Overview

Spring Boot 환경에서 AI API와 OCR 기능을 연동하고,
이미지로부터 텍스트를 추출하는 과정을 학습하기 위해 작성한 프로젝트입니다.

주요 학습 내용은 다음과 같습니다.

* Spring Boot 프로젝트 구성
* MyBatis 연동
* OCR 기능 구현
* 이미지 데이터 처리
* DTO / Service / Mapper 계층 구조 이해

---

## ⚙️ Tech Stack

* Java
* Spring Boot
* MyBatis
* AI / OCR
* Maven
